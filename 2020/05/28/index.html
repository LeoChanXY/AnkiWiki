正则中var xxx表示什么意思？P7

表示定义一个名称为xxx的变量
var regex 就是定义一个名为regex（正则表达式）的变量
var string 就是定义一个名为string（字符串）的变量

当进行模糊匹配时，具体到某一位字符时，怎么表示不确定的字符？P7

正则中的花括号和方括号分别表示什么？P7

花括号表示横向模糊匹配（长度）
方括号表示纵向模糊匹配（某个不确定的字符）

正则中横向模糊匹配用什么表示？P7

花括号表示横向模糊匹配（长度）（编码：花花肠子）

正则中纵向模糊匹配用什么表示？P7

方括号表示纵向模糊匹配（具体到某一位字符时，可以是多个字符中的某一个）（编码：放纵很多回）

横向模糊匹配中的花括号 与 纵向模糊匹配中的方括号，内部有什么区别？P7

横向模糊：花括号：两个值，中间逗号，表示范围从a到b（编码：花肚兜）
纵向模糊：方括号：多个值，中间没符号，表示abc中的某一个

模糊匹配有哪两种？P7

横向和纵向
